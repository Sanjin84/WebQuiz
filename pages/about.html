Our Mission

Our mission is to make it easy to host any coding competitions, CTFs, and other contests. These events are often run by volunteers, so we create free software to make their lives easier and their contests more fun.

Features

  • Question entry interface
  • GPT-4 question generator
  • Live scoreboard
  • Short answer and multiple-choice questions with images

How to Use

If you are a developer or a coding enthusiast, the beta is free to use:

https://github.com/Sanjin84/WebQuiz

https://github.com/Sanjin84/WebQuizAPI

Host the API on a cloud computer, and you are good to go.
